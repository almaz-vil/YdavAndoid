Удалённое управление телефоном

Как известно в условиях большой территории и холмистой местности сигнал сотовой связи значительно ослабевает. Стоит приподнять телефон на некоторую высоту и уровня сигнала сотовой связи может хватить для приёма и отправки СМС. Наша программа Ydav представляет собой сервер для управления телефоном с компьютера системой Windows 7-10 и Linux.
Принцип работы: телефон с сервером Ydav и клиент компьютер подключаются к одной сети Wi-Fi. После активации сервера появиться информация о его IP адресе и номер порта. В случае если телефон работает в режиме точки доступа, IP адрес сервера определяется на компьютере и является шлюзом. Сервер обеспечен текстовым протоколом TCP/IP формат сообщений JSON. Что позволяет общаться с ним через telnet. Безопасность Ваших данных зависть от защищенности Wi-Fi сети.

Протокол общения с сервером следующий:

  • Подключение
  • Запрос
  • Ответ
  • Отключение
Доступны следующие сообщения в формате JSON\n

Запрос для получения общей информации:

{"SendSMS":0,"SMSOUTInfo":0,"SMSOUTNomer":"","SMSOUTNomerInPS":"", "SMSOUTText":"","ReadINSMS":0,"ReadContact":0,"DeleteINSMS":""}
Описание структуры: 
Поля				Тип			Назначение
SendSMS				число		1-отправить СМС, 0 — по умолчанию
SMSOUTInfo			число		1- получить информацию об отправленных СМС, 0 — по умолчанию
SMSOUTNomer			строка		номер получателя СМС, пустая — по умолчанию
SMSOUTNomerInPS		строка		для Вашей идентификации СМС, пустая — по умолчанию
SMSOUTText			строка		текст СМС сообщения, пустая — по умолчанию
ReadINSMS			число		1-считать входящие СМС, 0 — по умолчанию
ReadContact			число		1-считать контакты, 0 — по умолчанию
DeleteINSMS			строка		удаление входящих СМС из базы сервера формата «N;N;”- номер СМС и точка с запятой: точка с запятой — по умолчанию

Ответ общий информации:

{"TIME":"вс февр. 02 12:05:08 GMT+03:00 2020","NEYWORKTYPE":0,"SIM":"ru","RSSI":137,"ASU":12,"ERROR":99,"CALL":"вызов завершен","NOMER":"","BATTERY_STATUS":"батарея в хорошем состоянии","LEVEL":"72","MAXLEVEL":"100","SMSInCount":0}
Описание структуры: 
Поля		Тип			Назначение
TIME		строка		текущие время на сервере
NEYWORKTYPE	число		текущий тип подключения к данным
SIM			строка		суффикс  страны текущей сотовой сети
RSSI		число		цифровая характеристика сотовой сети
ASU			число		цифровая характеристика сотовой сети
ERROR		число		цифровая характеристика сотовой сети
CALL		строка 		состояние телефонной линии
NOMER		строка		номер телефона входящего звонка
LEVEL		строка		заряд аккумулятора
MAXLEVEL	строка		максимальный заряд аккумулятора
SMSInCount	число		количество входящих СМС

Запрос для чтения контактов:

{"SendSMS":0,"SMSOUTInfo":0,"SMSOUTNomer":"","SMSOUTNomerInPS":"","SMSOUTText":"","ReadINSMS":0,"ReadContact":1,"DeleteINSMS":""}
Ответ все контакты с телефона в виде JSON массива структур. 
Описание структуры:
Поля		Тип			Назначение
ID			строка		уникальный идентификационный номер
NOMER		строка		номер телефона
NAME		строка		имя контакта, фамилия, имя и отчество

Запрос для входящих СМС, полученных при включенном сервере:

{"SendSMS":0,"SMSOUTInfo":0,"SMSOUTNomer":"","SMSOUTNomerInPS":"","SMSOUTText":"","ReadINSMS":1,"ReadContact":0,"DeleteINSMS":""}
Ответ все (см. поле SMSInCount ответ по умолчанию) входящие СМС с телефона в виде JSON массива структур. 
Описание структуры:
Поля			Тип		Назначение
ID			строка		уникальный идентификационный номер
NOMER		строка		номер телефона
TEXT			строка		текст сообщения
TIME			строка		время получения СМС

Запрос отправки СМС:

{"SendSMS":1,"SMSOUTInfo":0,"SMSOUTNomer":"900","SMSOUTNomerInPS":"1","SMSOUTText":"СПРАВКА","ReadINSMS":0,"ReadContact":0,"DeleteINSMS":""}

Запрос инфрмации об отправки СМС с индификатором "SMSOUTNomerInPS":"1":

{"SendSMS":0,"SMSOUTInfo":1,"SMSOUTNomer":"","SMSOUTNomerInPS":"1","SMSOUTText":"","ReadINSMS":0,"ReadContact":0,"DeleteINSMS":""}

Ответ сервера:

Описание структуры:
Поля			Тип		Назначение
NOMERsPK		строка		уникальный идентификационный номер
SENDInfo		строка		состояние отправки "-1" - не известно, "ОК" - отправлено
TEXT			строка		состояние доставки "-1" - не известно, "ОК" - доставлено
